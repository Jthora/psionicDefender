## 5) Immediate Tech Lab Experiments (Safety-Gated)
Each experiment has a measurable output and a hard stop condition.

### Experiment A: RF Shielding Layer Benchmark
- Objective: Quantify shielding effectiveness in dB across selected frequency bands.
- Protocol baseline: experiments/rf_shielding_tests/test_protocol.md
- Output: Update experiments/rf_shielding_tests/test_data.csv with per-layer and stacked-layer results.
- Safety gate: RF power capped to low-power bench levels; stop on unexpected heating or device instability.

### Experiment B: Acoustic Layer Transfer Function
- Objective: Measure attenuation vs frequency for foam, MLV, and composite stacks.
- Protocol baseline: experiments/acoustic_tests/test_methodology.md
- Output: Update experiments/acoustic_tests/test_results.csv with baseline and attenuated dB values.
- Safety gate: hearing protection mandatory for high SPL tests; stop if discomfort appears.

### Experiment C: Real-Time FFT Survey for Environmental Interference
- Objective: Build a repeatable interference profile in your Tech Lab using time-series FFT snapshots.
- Script baseline: software/dsp_module/frequency_analysis_script.py
- Output: Store file_analysis_results.csv snapshots and compare dominant peaks by time window.
- Safety gate: no ultrasonic/high-SPL emissions during occupied lab periods.

### Experiment D: Narrow-Band Noise Cancellation Characterization
- Objective: Validate notch filter depth and collateral distortion around target frequencies.
- Script baseline: software/dsp_module/noise_cancellation_code.py
- Output: Capture cancellation_log.csv and compute attenuation at target +/- bandwidth.
- Safety gate: start low gain; stop if clipping/feedback oscillation occurs.

### Experiment E: Passive Magnet Layout A/B Test
- Objective: Compare static magnet ring configurations for repeatable sensor-side effects (if any) rather than subjective perception.
- Build baseline: docs/quick_start_build.md
- Output: Structured A/B measurements with identical geometry except magnet arrangement.
- Safety gate: magnet handling controls from docs/safety_guidelines.md; keep away from storage media and medical implants.

### Experiment F: Reproducibility Run (Cross-Day Repeatability)
- Objective: Ensure effects observed in A-E are repeatable over multiple days and not one-off artifacts.
- Method: Repeat each test at least three times under logged environmental conditions.
- Output: Mean, standard deviation, and confidence interval for each metric.
- Safety gate: abort run on any equipment fault or uncontrolled temperature rise.

## 6) Decision Rule for What Goes Into Psi Defender v1
Promote an idea from hypothesis to build candidate only if all are true:
- Measurable metric exists and is logged.
- At least 3 independent repeats show similar effect size.
- Confounders are documented and bounded.
- Safety and legal constraints are satisfied.
